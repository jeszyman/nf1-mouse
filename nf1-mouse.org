** Project overview
K08 Aim 2: cfDNA monitoring of tumor biology and therapy response in /in vivo/ NF1 MPNST PDX models.
- Aim 2.1: Correlate cfDNA features with tumor characteristics at terminal bleed
- Aim 2.2: Characterize cfDNA dynamics in serial, non-terminal bleeds
- Aim 2.3: Infer tumor molecular changes from cfDNA dynamics
** Experimental system
- 4 PDX lines: JH-2-002, WU-356 (Chr8q-gain); JH-2-055, WU-487 (Chr8q-WT)
- Host strain: NOD.Cg-Rag1^{null} Il2rg^{null} (NRG)
- 3 treatment arms: vehicle, MEKi (mirdametinib 1.5 mg/kg PO QD), PLK1i (volasertib 5 mg/kg IP QOD)
- Serial submandibular bleeds (50–100 μL) at baseline and days 7, 14, 21, 28
- Terminal bleed (~400 μL) and tumor harvest at day 28
- cfDNA isolated with MagMax kit, profiled by EM-seq

** NF1 MPNST PDX model system
NF1-associated malignant peripheral nerve sheath tumors (MPNSTs) are aggressive sarcomas and the leading cause of cancer death in NF1. Patient-derived xenograft (PDX) models recapitulate tumor histology and molecular features.

- PDX lines:
  | Line      | Chr8q Status | Source Institution |
  |-----------+--------------+--------------------|
  | JH-2-002  | Gain         | Johns Hopkins      |
  | WU-356    | Gain         | WashU              |
  | JH-2-055  | WT           | Johns Hopkins      |
  | WU-487    | WT           | WashU              |
- Host strain: NOD.Cg-Rag1^{null} Il2rg^{null} (NRG), immune deficient
- Tumors implanted subcutaneously, early passage (P2–P4)
- Treatment begins when tumors reach 150–200 mm³
- 3 arms: vehicle, MEKi (mirdametinib), PLK1i (volasertib)
- n = 4 mice per line per arm (12 conditions, 48 mice total)
** cfDNA as translational platform
cfDNA from PDX mouse plasma provides a real-time, systemic measure of tumor biology:
- Serial submandibular bleeds enable longitudinal monitoring without sacrificing the animal
- Human-specific reads recovered after Disambiguate filtering
- Multi-omic cfDNA features: copy number (ichorCNA), fragmentomics (NNMF signatures), methylation (EM-seq), nucleosome positioning (Griffin)
- Terminal bleeds paired with matched tumor tissue for validation

#+begin_src yaml
# Mice
# Mice table contains mouse-specific metadata for research subjects
  # Primary key also serves as foreign key to subjects table
  # Includes strain, genotype, and housing information
  - name: mice
    title: Mouse Research Subjects
    description: Mouse subjects used in research
    path: data/nf1_manual_inputs.xlsx
    control:
      type: excel
      sheet: mice
    schema:
      missingValues: &mv ["", "unknown", "UNK", "unk", "n/a", "NA", "GET", "None"]
      fields:
        # PRIMARY KEY - also serves as foreign key to subjects table
        # Three-letter prefix: 'mou' + 4 digits
        - name: mouse_id
          title: Mouse Identifier
          description: Unique identifier for mouse subject. Format is 'mou' followed by 4 digits. Links directly to subject_id in subjects table.
          example: mou0001
          type: string
          constraints:
            required: true
            unique: true
            pattern: '^mou\d{4}$'

        # MOUSE-SPECIFIC FIELDS

        - name: strain
          title: Mouse Strain
          description: Genetic background strain of the mouse
          example: nod
          type: string
          constraints:
            required: true
            maxLength: 50

        - name: stock
          title: Mouse Stock
          description: Full stock designation including genetic modifications, lowercase with underscores
          example: nod_cg_rag1_null_il2rg_null
          type: string
          constraints:
            required: true
            maxLength: 100

        - name: sex
          title: Mouse Sex
          description: Biological sex of the mouse
          example: m
          type: string
          constraints:
            required: true
            enum: [m, f]
          enumDescriptions:
            m: Male
            f: Female

        - name: birth_date
          title: Birth Date
          description: Date of birth in ISO8601 format (YYYY-MM-DD)
          example: 2024-01-15
          type: date
          constraints:
            required: false
            minimum: "2020-01-01"
            maximum: "2026-12-31"

        - name: institution_housed
          title: Institution Housed
          description: The institution where the mouse was housed for experiments
          example: washu
          type: string
          constraints:
            enum: [washu, nci]
          enumDescriptions:
            washu: Washington University in St. Louis
            nci: National Cancer Institute

        - name: study_protocol_id
          title: Study Protocol
          description: Identifier for institution research protocol
          example: 99-CC-0168
          type: string
          constraints:
            required: false

        # OPTIONAL NOTES
        - name: notes
          title: Notes
          description: Free-text notes about this mouse
          type: string
          constraints:
            required: false
            maxLength: 500

    primaryKey: mouse_id
    foreignKeys:
      - fields: mouse_id
        reference:
          resource: subjects
          fields: subject_id
#+end_src
